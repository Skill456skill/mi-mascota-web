# 🐶🐱 Mi Mascota y Yo

**Mi Mascota y Yo** es un portal web diseñado para amantes de los animales. Ofrece servicios de registro de mascotas, tienda online, foro comunitario y consejos sobre el cuidado responsable de mascotas.
